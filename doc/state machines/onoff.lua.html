Hsm onoff.lua

On-floor detecting state machine.

This state machine is usefull for powering on and off the robot when placed on the floor or picked up. It uses the proximity sensor. To integrate, import t nd add behavior to the states.ON and states.OFF states. Also emits events.FLOOR and events.NOTFLOOR events.
For an example of a state mechine using this, see colorway.lua
