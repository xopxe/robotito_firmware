Module led_ring

LED ring.

Configuration is loaded using nvs.read("led_ring", parameter) calls, where the available parameters are:

  • "power" max power to use, in the 0..255 range. Defaults to 20

Functions

set_power (power) Set maximum power
clear ([r=0[, g=0[, b=0]]]) Sets all pixels.
set_led (led[, r=0[, g=0[, b=0]]]) Sets a LED to a specified color.
set_segment (segment[, r=0[, g=0[, b=0]]]) Controls a segment.
set_arc (led, length[, r=0[, g=0[, b=0]]]) Controls an arc.
update () Sends the pixel data to the device.

Fields

n_leds Number of LEDs in ring.


Functions

set_power (power)
Set maximum power

Parameters:

  • power In the 0..255 range
clear ([r=0[, g=0[, b=0]]])
Sets all pixels. This call is not limited by the power settings.

Parameters:

  • r integer red value in the 0..255 range. (default 0)
  • g integer green value in the 0..255 range. (default 0)
  • b integer blue value in the 0..255 range. (default 0)
set_led (led[, r=0[, g=0[, b=0]]])
Sets a LED to a specified color. The values will not be applied until update is called. This call is not limited by the power settings.

Parameters:

  • led Index in the 1..24 range
  • r integer red value in the 0..255 range. (default 0)
  • g integer green value in the 0..255 range. (default 0)
  • b integer blue value in the 0..255 range. (default 0)
set_segment (segment[, r=0[, g=0[, b=0]]])
Controls a segment. The 6 segments can be assigned a color.

Parameters:

  • segment Index in the 1..6 range.
  • r integer or boolean red value in the 0..255 range. If r is true then a predefined r,g,b color will be used. If false, the segment will be switched off. (default 0)
  • g integer green value in the 0..255 range. (default 0)
  • b integer blue value in the 0..255 range. (default 0)
set_arc (led, length[, r=0[, g=0[, b=0]]])
Controls an arc. Paint a set of consecutive leds.

Parameters:

  • led integer Index in the 1..24 range.
  • length integer number of consecutive leds to paint -24..24 range.
  • r integer red value in the 0..255 range. (default 0)
  • g integer green value in the 0..255 range. (default 0)
  • b integer blue value in the 0..255 range. (default 0)
update ()
Sends the pixel data to the device. This is to be used with set_led.
