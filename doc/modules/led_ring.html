Functions

set_power (power)
Set maximum power

Parameters:

  • power In the 0..255 range
clear (r, g, b)
Sets all pixels. This call is not limited by the power settings.

Parameters:

  • r red value in the 0..255 range. Defaults to 0.
  • g green value in the 0..255 range. Defaults to 0.
  • b blue value in the 0..255 range. Defaults to 0.
set_led (led, r, g, b)
Sets a LED to a specified color. The values will not be applied until update is called. This call is not limited by the power settings.

Parameters:

  • led Index in the 1..24 range
  • r red value in the 0..255 range. Defaults to 0.
  • g green value in the 0..255 range. Defaults to 0.
  • b blue value in the 0..255 range. Defaults to 0.
set_segment (segment, enable)
Controls a segment. The 6 segment have a predefined color assigned.

Parameters:

  • segment Index in the 1..6 range.
  • enable true value to enable, false to disable.
set_segment_rgb (segment, r, g, b)
Controls a segment. The 6 segments can be assigned a color.

Parameters:

  • segment Index in the 1..6 range.
  • r red value in the 0..255 range. Defaults to 0.
  • g green value in the 0..255 range. Defaults to 0.
  • b blue value in the 0..255 range. Defaults to 0.
update ()
Sends the pixel data to the device. This is to be used with set_led.
init (power)
Initialization. This configures the LED ring.

Parameters:

  • power max power to use, in the 0..255 range. If nor provided, will be read from nvs.read("led_ring","power"), defaults to 20.

Fields

n_leds
Number of LEDs in ring.
