Set maximum power

Parameters:

  • power In the 0..255 range
clear ()
Switch off all pixels.
set_led (led, r, g, b)
Sets a LED to a specified color. The values will not be applied until update is called.

Parameters:

  • led Index in the 1..24 range
  • r red value in the 0..255 range. Defaults to 0.
  • g green value in the 0..255 range. Defaults to 0.
  • b blue value in the 0..255 range. Defaults to 0.
update ()
Sends the pixel data to the device. This is to be used with set_led.
init (power)
Initialization. This configures the LED ring.

Parameters:

  • power max power to use, in the 0..255 range. If nor ptovided, will be read from nvs.read("led_ring","power"), deafults to 20.
