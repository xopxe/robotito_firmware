Module proximity

Downward facing proximity sensor.

This modeule is used to detect when the robot is picked up.

Functions

enable (on[, period=100[, threshold=250[, hysteresis=3]]])
Enables the proximity callback. When enabled, proximity changes will trigger cb.

Parameters:

  • on boolean true value to enable, false value to disable.
  • period integer Sampling period in ms, if omitted is read from nvs.read("proximity","period"). (default 100)
  • threshold integer proximity reference value, if omitted is read from nvs.read("proximity","threshold") (default 250)
  • hysteresis integer if omitted is read from nvs.read("proximity","hysteresis") (default 3)

Fields

device
The native C firmware module. This can be used to access low level functionality from apds9969.proximity. FIXME: docs
cb
The callback module for the proximity sensor. This is a callback list attached to the proximity sensor, see cb_list. This call triggers on threshold crossing, see enable. The parameter of the callback is a boolean which is true when the object is close.

Usage:

    local local proximity = require 'proximity'
proximity.cb.append( function (v) print("too close:", v) end )
