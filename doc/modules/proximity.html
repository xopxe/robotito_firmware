Module proximity

Downward facing proximity sensor.

This modeule is used to detect when the robot is picked up.

Functions

M.threshold.enable (on, period, threshold, hysteresis)
Enables the proximity threshold callback. When enabled, proximity changes will trigger M.threshold.cb.

Parameters:

  • on true value to enable, false value to disable.
  • period Sampling period in ms, if omitted is read from nvs.read("proximity_sensor","period"), defaults to 100.
  • threshold proximity reference value, if omitted is read from nvs.read("proximity_sensor","threshold") (defaults to 250, about 2cm)
  • hysteresis if omitted is read from nvs.read("proximity_sensor","hysteresis") (defaults to 3)

Fields

device
The native C firmware module. This can be used to access low level functionality from apds9969.proximity. FIXME: docs
M.threshold.cb
The callback module for the proximity sensor. This is a callback list attached to the proximity sensor, see cb_list. This call triggers on threshold crossing, see M.threshold.enable. The parameter of the callback is a boolean which is true when the object is close.
  • v true if distance is greater than threshold, false otherwise.

Usage:

    local local proximity = require 'proximity'
proximity.threshold.cb.append( function (v) print("close:", v) end )
