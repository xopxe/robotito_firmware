Module cb_list

List of functions to be used as callback.

When you have an API that offers a callback, but it must be used from many places. You register this as callback, and then anyone can add itself to the list of callees.

Usage:

    local cb_list = require'cb_list'.get_list()
some_lib.some_register_callback(cb_list.call)
cb_list.append( function() print "call!" end )

Functions

cb.append (f) Append a function to the list of callbacks.
cb.remove (f) Remove a function from the list of callbacks.
cb.call (...) The function to be passed to the API requesting a callback function.


Functions

cb.append (f)
Append a function to the list of callbacks. The function will be called with the parameters provided to the callback.

Parameters:

  • f a function to be called.
cb.remove (f)
Remove a function from the list of callbacks. The function will be called with the parameters provided to the callback.

Parameters:

  • f the function to be removed.
cb.call (...)
The function to be passed to the API requesting a callback function. Invoking this function will call all the functions registered using append

Parameters:

  • ... parameters will be passed to each function. Returns will be ignored.
