Module robot

Robot module.

This is the main module for the robot. Contains all the services available. All the services are already initalized using data stored in non-volatile storage. Check each module's documentaion to see the used variables.

Fields

id Unique ID of the robot.
omni omni-directional mobility platform.
color Downard facing color sensor.
floor Downard facing proximity sensor.
laser_ring Distance sensor ring.
led_ring UI LED ring.
wifi_net WiFi UDP network susbsystem.


Fields

id
Unique ID of the robot. Read from nvs.read("robot","id"), defaults to 0.

omni-directional mobility platform.

UI LED ring. This points to led_ring
