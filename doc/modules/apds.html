Module apds

Module for the apds9960 color/proximity/gesture sensor.

Color definitions and setings are loaded using nvs.read("color_sensor", parameter) calls, where the available parameters are:

  • "min_sat" If the saturaion is below this, color is 'unknown' (default is 24)
  • "min_val" If the value is below this, color is 'black' (default is 40)
  • "max_val" If the value is above this, color is 'white' (default is 270)
  • "min_h_red", "max_h_red" Hue range for 'red' (default is 351, 359)
  • "min_h_yellow", "max_h_yellow" Hue range for 'yellow' (default is 22, 65)
  • "min_h_green", "max_h_green" ` Hue range for 'green' (default is 159, 180)
  • "min_h_blue", "max_h_blue" ` Hue range for 'blue' (default is 209, 215)
  • "min_h_magenta", "max_h_magenta" ` Hue range for 'magenta' (default is 255, 300)

Functions

M.proximity.threshold.enable (on, period, threshold, hysteresis)
Enables the proximity threshold callback. When enabled, proximity changes will trigger M.proximity.threshold.cb.

Parameters:

  • on true value to enable, false value to disable.
  • period Sampling period in ms, if omitted is read from nvs.read("proximity_sensor","period"), deafults to 100.
  • threshold proximity reference value, if omitted is read from nvs.read("proximity_sensor","threshold") (deafults to 250, about 2cm)
  • hysteresis if omitted is read from nvs.read("proximity_sensor","hysteresis") (deafults to 3)
M.color.light (on)
Control the LED light for the color sensor.

Parameters:

  • on true to switch on, false to switch off.
M.color.change.enable (on, period)
Enables the color change callback. When enabled, color changes will trigger M.color.change.cb.

Parameters:

  • on true value to enable, false value to disable.
  • period Sampling period in ms, if omitted is read from nvs.read("color_sensor","period"), deafults to 100.
M.color.continuous.enable (on, hsv)
Enables the color monitoring callback.
See M.color.continuous.cb. The period in ms to use is read from nvs.read("color_sensor","period"), deafults to 100.

Parameters:

  • on true value to enable, false value to disable.
  • hsv true to set HSV mode (see M.color.continuous.cb)

M.proximity.threshold.cb
The callback module for the proximity sensor. This is a callback list attached to the proximity sensor, see cb_list. This call triggers on threshold crossing, see M.proximity.threshold.enable. The parameter of the callback is a boolean which is true when the object is close.
  • v true if distance is greater than threshold, false otherwise.

Usage:

    local local apds = require 'apds'
apds.proximity.threshold.cb.append( function (v) print("close:", v) end )
M.color.device
The native C firmware module. This can be used to access low level functionality from apds9969.color. FIXME: docs
M.color.change.cb
The callback module for the color change sensor. This is a callback list attached to the color sensor, see cb_list. The callback will be called with (color, s, v)
* color: one of "red", "yellow", "green", "blue", "magenta", "black", "white", "unknown"
* s,v: 0..255

Usage:

    local local apds = require'apds'
apds.color.change.cb.append( function (color, s, v) print(color, s, v) end )
M.color.continuous.cb
The callback module for the color monitoring sensor. This is a callback list attached to the color sensor, see cb_list. The callback will be called with (r,g,b,a [,h,s,v, color])
hsv will be provided if HSV mode enabled (see M.color.continuous.enable)
* r,g,b,a : 16 bits
* h: 0..360
* s,v: 0..255
* color: one of "red", "yellow", "green", "blue", "magenta", "black", "white", "unknown"

Usage:

    local local apds = require'apds'
apds.color.continuous.cb.append( function (r, g, b, a) print(r, g, b, a) end )
